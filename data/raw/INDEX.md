# data/raw — 원시 데이터 인덱스 (대두유 조달 분석 Phase A)

**작성일**: 2026-06-22  
**패널 역할**: C-02(시장조사) · C-03(데이터 파이프라인) · P1-01(상품) · P1-02(지정학) · P1-03(기후) · P1-04(공급망)  
**범위**: 2017~2026년 국가별·월별 대두유 공급·수요·가격 영향 요인 데이터 전체

---

## 폴더 구조 개요

```
data/raw/
├── USDA/
│   ├── FAS/
│   │   ├── GAIN/
│   │   │   ├── Biofuels/   ← 바이오연료 정책 보고서 (2017~2026, 280+개)
│   │   │   └── Oilseeds/   ← 유지작물 공급·수요 보고서 (2017~2026, 950+개)
│   │   ├── GATS/
│   │   │   ├── 1507.10/    ← 미국 조대두유(HS1507.10) 수출량·재수출량
│   │   │   ├── 1507.90/    ← 미국 정제대두유(HS1507.90) 수출량
│   │   │   └── export_value_top10/  ← 9개년 수출액(USD) 상위 10개국
│   │   └── PSD/            ← USDA FAS PSD 공급·수요 전망 xlsx
│   └── WASDE/              ← USDA WASDE 월간 보고서 xlsx (2017~2026)
└── FAO/
    └── AMIS/               ← FAO AMIS 식용유 시장 정보
```
