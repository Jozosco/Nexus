# USDA FAS GAIN — Oilseeds Reports

USDA Foreign Agricultural Service의 GAIN(Global Agricultural Information
Network)에서 발행한 Oilseeds 관련 보고서 PDF 모음.

- 파일명 규칙: `YY.MM_Country_Title.pdf` (YY.MM = 발행 연월, Country = 발행 대상국)
- 기간: 2017년 ~ (수동 수집분)
- 대두유 활용: 작황 전망·정책 변경(수입관세/수출세)·재고전망·무역흐름 서술 추출
  → G1 변수 보조 신호(WASDE 서프라이즈 선행, GPR 보조)
- 파싱 도구(예정): scripts/ingest_gain_oilseeds_pdf.py (pdfplumber 기반)
