# USDA FAS GAIN — Biofuels Reports

USDA Foreign Agricultural Service GAIN(Global Agricultural Information
Network) 발행 Biofuels 관련 보고서 PDF 모음.

- 파일명 규칙: `YY.MM_Country_Title.pdf` (YY.MM = 발행 연월, Country = 발행 대상국)
- 분류 기준: 2020-09~12 업로드분 + biofuel/biodiesel/ethanol/renewable 주제 키워드
- 대두유 활용: 바이오디젤·SAF 수요 정책(RVO/EU RED) → 식용유↔연료 수요 전환 신호
  → G2 변동성 레이어 보조 입력 (식용유 수요 채널)
