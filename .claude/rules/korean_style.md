## §1 문장 톤 & 격식

| 상황 | 사용할 격식 | 예시 |
|---|---|---|
| 보고서 본문 | 전문 서술체 (`~임`, `~함`, `~됨`) | "글로벌 공급 부족이 심화됨." |
| 사용자 응답 | 합쇼체 (`~합니다`, `~입니다`) | "분석이 완료되었습니다." |
| 경고/알림 메시지 | 해라체 간결 (`[경고]`, `[오류]`) | "[경고] 데이터 없음." |
| 코드 주석 | 간결 한국어 (~임, ~함) | `# 대두유 행 탐색` |

## §2 번역 투 방지 규칙

### 2.1 피해야 할 표현 → 자연스러운 대안

| 피해야 할 (번역 투) | 자연스러운 한국어 |
|---|---|
| "~의 경우에는" | "~이면" / "~일 때" |
| "~에 대한 분석을 수행하였음" | "~을 분석함" |
| "~임을 확인할 수 있음" | "~임" |
| "데이터가 존재하지 않음" | "데이터 없음" |
| "~을 통해 도출됨" | "~으로 산출됨" / "~에서 도출됨" |
| "수행하다" (make, perform 직역) | "실행하다", "진행하다", "처리하다" |
| "~에 대하여" | "~에 관해" / "~을" |
| "~할 수 있는 것으로 판단됨" | "~으로 보임" / "~가 예상됨" |

### 2.2 숫자·단위 표기

- 천 단위 구분: `1,000 MT` (숫자 + 단위 사이 공백)
- 퍼센트: `10%` (공백 없음)
- 연도 범위: `2017~2025년` (물결표 `~`, 마지막에만 `년`)
- 날짜: `2026년 6월 18일` 또는 `2026-06-18` (ISO 형식 혼용 가능)

### 2.3 전문 용어 통일 (대두유 조달 도메인)

| 영어 | 한국어 표준 표기 |

| Refined Soybean Oil | 정제 대두유 |

## §3 보고서 구조 관행

### 3.1 섹션 제목
- 1단계: `##` — 대주제 (예: `## 분석 결과`)
- 2단계: `###` — 소주제 (예: `### 상위 5개 핵심 변수`)
- 번호 없이 자연스럽게 (목차가 있는 경우만 번호 사용)

### 3.2 수치 제시 방식
```
# 좋은 예
BDI가 90일 이동 평균 대비 2σ(표준편차)를 초과하면 해운 비용 급등 경보를 발령함.

# 나쁜 예 (번역 투)
BDI가 90일 이동 평균 대비 2σ를 초과하는 경우에는 해운 비용 급등에 대한 경보가 발령되어야 함.
```

### 3.3 인과관계 서술 패턴 (G1 보고서용)
단계별 인과 체인은 → 기호로 연결하고, 각 단계를 명사형으로 마무리:

```
원인 발생 → 중간 메커니즘 → 가격 영향 방향
예: 엘니뇨 강세 → 남미 대두 감산 → 글로벌 압착량 감소 → SBO 공급 축소 → 가격 상승 압력
```
